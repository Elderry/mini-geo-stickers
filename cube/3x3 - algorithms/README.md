# 3x3 Algorithms

This folder contains compact SVG references for selected 3x3 last-layer algorithms. Each pattern is drawn from the top face: the grid shows the last-layer sticker layout, the small bars around the edge show side-facing last-layer stickers, and arrows show the piece cycle for permutation cases.

The steps use standard cube notation. A letter turns that face clockwise, `'` turns it counterclockwise, `2` means a half turn, lowercase moves are wide turns, and `x` is a cube rotation.
